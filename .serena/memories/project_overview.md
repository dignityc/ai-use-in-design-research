# Paper Review with LLM - 프로젝트 개요

## 프로젝트 목적
논문 리뷰 자동화를 위한 Python 스크립트 모음. 데이터 분석 및 시각화를 통해 논문 연구 생태계를 체계적으로 탐색하는 워크플로우 제공.

## 메인 워크플로우
1. **분석워크 (analysis_work/)**: 7-13단계의 NLP 분석 파이프라인
   - 데이터 정규화 → 토픽 모델링 → 대시보드 생성 → 시맨틱 네트워크 분석
2. **코딩워크 (coding_work/)**: Google Sheets API를 통한 논문 분류 자동화

## 데이터 규모
- **소스 데이터**: 511개 논문 리스트 → 93개 분석 완료
- **코딩워크**: 900개 논문 × 18개 컬럼 (525개 분류 완료, 58.3%)

## 주요 성과물
- **인터랙티브 대시보드**: 단일 HTML 파일 (38KB)로 모든 분석 결과 탐색
- **BERTopic 분석**: 9개 주요 토픽 발견 및 AI 방법론 진화 추적
- **시맨틱 네트워크**: 논문 간 의미론적 관계 네트워크 구축

## 프로젝트 특징
- **단계별 워크플로우**: 각 분석 단계가 독립적으로 실행 가능
- **결과물 중심**: 각 단계마다 구체적인 output 생성
- **재현 가능성**: 모든 분석 과정이 스크립트로 자동화됨
